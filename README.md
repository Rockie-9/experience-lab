# 體驗實驗室 · Experience Lab

> **每個星期五,打開一個新的自我——把日常最不起眼的角落,變成只有你能寫的田野筆記。**

## 為什麼這個實驗

這不是挑戰、不是打卡、不是 30 天計畫。

這是一份給自己、一年 52 次、被認真觀察的機會。每題都從日常最熟悉的角落出發——衣櫃、書架、通勤、那部重看過的電影、凌晨三點才敢點開的歌單——用兩位有正當性的專家輪流訪問,擠出單一視角看不到的立體感。

年底你會擁有一份比日記更立體、比履歷更誠實的自我檔案。

## 三個 DNA

**DNA 01 · 雙專家對話**——兩個領域交叉訪問,像一面多角度的鏡子。
**DNA 02 · 三步就能開始**——每題最多三個操作步驟,手機就能做、廚房也能做。
**DNA 03 · 週五的儀式**——週五釋出、週末完成,你需要的不是紀律,是節奏。
